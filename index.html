What is Sacrament Hub?

Sacrament Hub is a modern web application that streamlines parish sacramental preparation. It replaces spreadsheets and manual processes with a unified platform for registration, enrollment, attendance, events, resources, and communication. Staff get a powerful administrative dashboard, while families and candidates receive a simple, guided experience.

Architecture Overview

  • Static export — no Node server in production, the entire app ships as static files on a global CDN for fast, reliable delivery.
  • Firebase backend — Auth, Firestore, Storage, and Cloud Functions power all server‑side logic, security, and data workflows.
  • Role‑based routing — staff and member experiences live in separate route groups, with a built‑in switcher for users who serve both roles.
  • Term & program scoping — nearly every screen is automatically filtered by the active term and program, driven from one shared context.
  • Real‑time sync — Firestore listeners keep dashboards, attendance, and notifications live across devices with no refresh required.
  • Offline‑first PWA — a service worker caches the app shell for offline use. New deploys never force a disruptive reload mid‑session, instead users get a gentle "update available" prompt so in‑progress work is never lost.
